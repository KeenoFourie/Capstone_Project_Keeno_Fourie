Router Descriptions

Home

HTTP Methods Paths Descriptions
GET /home This router takes you to the Home Page of the API.

Users

HTTP Methods Paths Descriptions
GET /users This router shows all the users in the database.
GET /user/:id This router shows a single user when a user requests an ID.
POST /user This router allows us to register/add a new user.
POST /login This router allows a user to login.
PATCH /user/:id This router allows us to update a users details in the database.
DELETE /user/:id This router allows us to delete a user and their details in the database.

Courses

HTTP Methods Paths Descriptions
GET /courses This router shows all the courses in the database.
GET /course/:id This router shows a single course when a user requests the course by ID.
POST /course This router allows us to add a new course into the database.
PACTH /course/:id This router is used to update a course and its details on the database.
DELETE /course/:id This router is used to delete a course from the database.
